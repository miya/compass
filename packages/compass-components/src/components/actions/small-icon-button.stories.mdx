# Small Icon Button

The SmallIconButton component is a compact button that displays an icon.
It's designed for use in tight spaces where a full button would be too large.

## Key features

- Icon-only display
- Multiple size options
- Disabled state support
- Click handler
- Dark mode support

## Usage

The SmallIconButton component is perfect for:

- Toolbars and action bars
- Compact interfaces
- Icon-only actions
- Space-constrained layouts

```tsx
import { SmallIconButton } from './small-icon-button';
import { Icon } from '../leafygreen';

// Basic usage
<SmallIconButton
  glyph={<Icon glyph="Play" />}
  label="Play"
  onClick={() => console.log('Play clicked')}
/>

// Different sizes
<SmallIconButton
  glyph={<Icon glyph="Play" />}
  label="Play"
  size="small"
/>

<SmallIconButton
  glyph={<Icon glyph="Play" />}
  label="Play"
  size="xsmall"
/>

// Disabled state
<SmallIconButton
  glyph={<Icon glyph="Play" />}
  label="Play"
  disabled
/>
```

## Best Practices

1. **Icon Selection**:

   - Use clear, recognizable icons
   - Ensure icons are meaningful in context
   - Consider using standard icon sets
   - Maintain consistency across similar actions

2. **Size Usage**:

   - Use default size for most cases
   - Use small size for secondary actions
   - Use xsmall size for very tight spaces
   - Maintain consistent sizing within groups

3. **Accessibility**:
   - Always provide a label
   - Ensure sufficient contrast
   - Support keyboard navigation
   - Include hover and focus states
